### 🚀 PROJETO
O Convert é um conversor de moeda online que visa converter os seguintes valores em Real:
* USD - Dolar;
* EUR - Euro;
* GBP - Libra;
Este projeto foi feito junto da aula Convert do módulo de Javascript da trilha Fullstack da Rocketseat.
